Ejercicios para entregar

Lenguaje de Marcas 2018/2019

01_texto_estructurado.html
Escribe una pequeña página en la que tengas: cabecera h1, título, texto bien organizado con etiquetas. Usa alguna entidad tipo © (&copy;) Puedes usar los formatos especiales de los ejercicios 5 a 7. La página debe pasar el validador y estar en el servidor el martes 2
02_texto_maquetado.html
Escribe algo similar al primer ejericio (puedes reutilizarlo si quieres ) y añade un fichero externo con teglas CSS para maquetar texto y ajustar las cajas con padding, border.... Usa 02_texto_maquetado.css como nombre del fichero de estilo.
03_horario.html
Escribe un documento con tu horario de clase de este curso. La tabla presentará una cabecera horizontal con los días de la semana y una cabecera en la primera columna con intervalors horarios del tipo 15:00-16:00. En el intervalo 18:00-18:30 aparecerá una celda con la palabra recreo que ocupe todo el ancho de tabla. Los módulos aparecerán con su nombre completo y si son sesiones de más de una hora, la celda correspondiente ocupará varias filas. Utiliza clases CSS para marcar cada módulo de su color correspondiente.
04_sudoku.html
Bájate el ejercicio del servidor.
05_calendario.html
Bájate el calendario de la carpeta descargas. Añade clases o identificadores a determinadas celdas y proporciona un css adecuado.
06_tarta.html
Busca en el INE o el Eustat o Opendata Euskadi información de algo que te interese y genera un gráfico con esos datos
07_tabla.html
Busca en el INE o el Eustat o Opendata Euskadi información de algo que te interese y genera una tabla con esos datos. Recuerda el truco para resaltar la línea en la que está el cursor. Puedes hacer una cebra...
08_galeria.html
Galería de fotos con un máximo de 100 fotos. Se pie una galería líquida; es decir que compacte los thumbnails en las columnas necesarias y los reorganice al modificar el tamaño de página.
09_formulario.html
Codifica un formulario de registro de un sitio web o similar utilizando campos de HTML4. Debes poner en action "http://10.22.23.71/~juanjo/00_primero/recoger_datos.php". No importa que el formulario no tenga demasiado sentido si la razón es que quieres usar muchos campos de tipos distintos. No hace falta que lo maquetes.
10_formularioHTML5.html
Formulario con distintos de los nuevos campos en HTML5 y además uso de regex en el atributo pattern para validar campos.
11_fuente
Debería ser una carpeta que contenga documentos y ficheros de fuentes. Crea uno o varios documentos HTML que usen fuentes de fichero o de googlefonts. Haz también alguna prueba con font awsome
12_instrucciones
Haz uso de Syntax Highlighter para generar un documento de instrucciones: por ejemplo: ¿Cómo se genera y da estilo a una tabla en HTML?
13_Contrubuir
Contribuye con algo al documento compartido que recopila lo que podemos embeber en una página web
14_Map
Usa alguna herramienta online para generar un mapa a partir de una imagen.
15_sprite
Busca una imagen que agrupe varias imágenes o iconos (un sprite), por ejemplo la del ahorcado. Utilizala para mostrar en un document sólo una de las imágenes o todas pero de una en una. Cuando termines busca iconos en internet y usa un generador de sprites para que los una en una sóla imagen y genere el CSS adecuado
16_datosxml
Busca en opendata euskadi o similar, algún fichero que te interese. Asóciale una hoja de estilo css para que se puedan ver los datos.
17_rss.xml
Mostrar mediante CSS el contenido de un canl de sindicación RSS
18_calendario.xml
Mostrar mediante XSLT las fiestas de Bilbao (+Bizkaia+CAV) del año que viene mediante un xslt. Investiga cómo se deberían escribir los fichero XSLT para que pase el validador. Intenta añadirle una hoja de estilo CSS
19_rss.xml
Mostrar mediante XSLT las cinco primeras noticias de un canal RSS. Añade una hoja de estilo. Transforma la información en enlaces
